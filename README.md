# PicturesRecycler


![](demo/PicturesRecycler.gif)

Purpose of this project is to do an app that displays random images via API.

Libraries:
- MVVM Architecture
- Retrofit2 & Gson
- Architecture components (LiveData, ViewModel, DataBinding)
- Glide
- RxJava

API Services used:
- Lorem Picsum (https://picsum.photos/)
